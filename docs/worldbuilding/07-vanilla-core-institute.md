# 香草芯研究所

## 全称与简称

- **正式名称**：香草芯植物精灵联合研究所
- **简称**：香草芯研究所 / 香草芯所
- **对外品牌**：与团队名 **香草芯 (Vanilla Xin)** 一致，所徽为心形花蕊与六叶茶纹（视觉可后补）

## 成立背景

| 时间         | 事件                                |
|------------|-----------------------------------|
| 谐纪 68 年    | 绿芯计划立项，应对谐露生态波动                   |
| 谐纪 72 年    | 初代植物精灵在共鸣区实验室「苏醒」，被赋予代号 **V-000** |
| 谐纪 75～78 年 | 以精灵之种分批培育 V-001～V-006             |
| 谐纪 80 年迄今  | 成为城邦「人灵共生」标杆单位，亦承受舆论与政治压力         |

## 使命（公开版）

1. 稳定区域谐露循环，预警露灾风险。
2. 培育植物精灵，探索与人社会的长期共生协议。
3. 将研究成果应用于魔法农业、医疗谐露与低损耗转换技术。

## 使命（档案未公开部分）

- 曾参与裂相纪末露灾区的**善后采样**（与种子来源悬念相关）。
- 持有 M3 级魔法实验许可与少量遗物研究只读权限（受统辖局审计）。【待展开】

## 组织（建议）

```
所长（人类，外聘行政）
├── 绿芯实验室 — 初代 & 茶系香草酱主力
├── 谐露工程组 — 人类 + 半灵技术员
├── 共生伦理委员会 — 法律与精灵权益
└── 外勤监测队 — 茶系轮值野外节点
```

- **初代**：精神领袖 + 绿芯实验室实际负责人（人类所长尊重其专业意见）。
- **六位后来者**：正式成员，非「实验品」；对外宣传为「香草酱」形象大使。

## 精灵之种与培育流程（设定草案）

1. **采集/获得** 精灵之种（来源：【待展开】）。
2. 种胚浸入**谐露孵化基质**，注入初代分化魔力（血脉共鸣）。
3. 意识成形后，按茶种植物特性赋予**元素亲和**与性格基底。
4. 通过「共生契约仪式」登记公民身份（谐纪 76 年起合法化流程）。
