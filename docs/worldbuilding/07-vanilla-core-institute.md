# 香草芯研究所

## 全称与简称

- **正式名称**：香草芯植物精灵联合研究所
- **简称**：香草芯研究所 / 香草芯

> 研究所是虚构机构；「源质 / 衍能 / 源历」等术语不因机构名而命名。

## 成立背景

| 时间         | 事件                    |
|------------|-----------------------|
| 源历 68 年    | 绿芯计划立项，应对源质生态波动       |
| 源历 72 年    | 初代苏醒，代号 **香草**（所内芯酱）  |
| 源历 75～78 年 | 以精灵之种培育六名茶系香草酱（黑茶～绿茶） |
| 源历 80 年迄今  | 人灵共生标杆单位              |

## 使命（公开版）

1. 稳定区域源质循环，预警质灾。
2. 培育植物精灵，探索共生协议。
3. 魔法农业、医疗源质、**降低转电损耗**相关研究。

## 组织（建议）

```
所长（人类，外聘行政）
├── 绿芯实验室 — 芯酱 & 六名茶系香草酱
├── 源质工程组 — 人类 + 半灵技术员
├── 共生伦理委员会
└── 外勤监测队
```

## 所内氛围：同事、朋友，偶尔像自家孩子

香草酱所内是**成员**，不是「摆在高处的样品」。人类职员、半灵技术员与她们的关系**融洽**
，日常能开玩笑、搭把手、下班顺路买饮料——更像处得好的同事与朋友，而不是客客气气的「家属探望」。

| 常见相处方式      | 表现                                     |
|-------------|----------------------------------------|
| **朋友 / 搭档** | 一起熬夜改报告、吐槽转电效率、绿茶酱跟工程组技术员捣鼓手环          |
| **平等同事**    | 工作上有分工，但讨论方案时会让香草酱发表意见，不摆长辈架子          |
| **偶尔的长辈感**  | 有人爱留零食、有人爱念叨安全守则——出于关心，**不是**全员把她们当小孩养 |
| **对芯酱**     | 同样尊重；有人叫她芯酱姐、有人叫香草老师，语气亲近但不疏远          |

- 称呼仍用「绿茶酱」等所内简称；对外公文用代号。
- 所内也会说「你家红茶酱又闯祸了」，但下一句往往是「过来，我帮你改」——**护短与并肩**，不是隔阂式的宠爱。
- 芯酱在绿芯实验室是**妈妈**（对六酱）与**负责人**（对工作）；六酱彼此之间是姐妹。这与职员和香草酱的关系**不是同一层**
  ——职员不必也不应强行当「家人」。
