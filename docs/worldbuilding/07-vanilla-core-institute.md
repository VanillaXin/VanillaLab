# 香草芯研究所

## 全称与简称

- **正式名称**：香草芯植物精灵联合研究所
- **简称**：香草芯研究所 / 香草芯

> 研究所是虚构机构；「源质 / 衍能 / 源历」等术语不因机构名而命名。

## 成立背景

| 时间         | 事件                 |
|------------|--------------------|
| 源历 68 年    | 绿芯计划立项，应对源质生态波动    |
| 源历 72 年 | 初代苏醒，代号 **香草**（所内芯酱） |
| 源历 75～78 年 | 以精灵之种培育六名茶系香草酱（黑茶～绿茶） |
| 源历 80 年迄今  | 人灵共生标杆单位           |

## 使命（公开版）

1. 稳定区域源质循环，预警质灾。
2. 培育植物精灵，探索共生协议。
3. 魔法农业、医疗源质、**降低转电损耗**相关研究。

## 组织（建议）

```
所长（人类，外聘行政）
├── 绿芯实验室 — 初代 & 茶系香草酱
├── 源质工程组 — 人类 + 半灵技术员（转衍/并网相关）
├── 共生伦理委员会
└── 外勤监测队
```
