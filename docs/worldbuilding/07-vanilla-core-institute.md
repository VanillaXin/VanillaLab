# 香草芯研究所 · 绿芯院区（主叙事）

> 世界上有多家香草芯研究所，上级为**香草芯研究总署**（见 [08-vanilla-core-network.md](08-vanilla-core-network.md)）。*
*本文件仅写绿芯院区**——当前全部香草酱主线发生于此。

## 全称与简称

| 语境   | 名称                    |
|------|-----------------------|
| 公文全称 | 香草芯植物精灵联合研究所·**绿芯院区** |
| 日常简称 | 香草芯研究所、所里、**绿芯院**     |
| 对外通称 | 常被媒体直接叫「香草芯研究所」（默认绿芯） |

## 院区背景

| 时间         | 事件                       |
|------------|--------------------------|
| 源历 68 年    | **绿芯计划**在本院立项（后纳入总署重点专项） |
| 源历 72 年    | 初代苏醒，代号 **香草**（所内芯酱）     |
| 源历 75～78 年 | 培育六名茶系香草酱（黑茶～绿茶）         |
| 源历 80 年迄今  | 总署辖下标杆院区；人灵共生宣传样板        |

## 使命（本院公开版）

1. 稳定**绿芯共鸣区**源质循环，预警质灾。
2. 培育植物精灵，探索共生协议（茶系香草酱为现阶段成果）。
3. 魔法农业、医疗源质、降低转电损耗——成果上报总署，参与**跨院区经验交流**（评比偏改进导向，非恶性竞争）。

## 本院组织

```
院长（人类，外聘；对总署负责）
├── 绿芯实验室 — 芯酱 & 六名茶系香草酱
├── 源质工程组 — 人类 + 半灵技术员
├── 共生伦理委员会（本院分支）
└── 外勤监测队
```

- 重大事务需向**总署**报备（种子使用、M3 级以上实验、精灵公民事项）。
- 他院区人员短期驻留时，所内仍称「X 院来的同事」，与香草酱可做朋友式相处。

## 所内氛围

香草酱是**本院成员**。人类职员、半灵技术员与她们**融洽**：多为同事与朋友，能开玩笑、并肩改方案；个别人会留零食、念叨安全，**没有
**家人式客套或隔阂。

| 常见相处方式      | 表现                     |
|-------------|------------------------|
| **朋友 / 搭档** | 熬夜改报告、吐槽转电、绿茶酱与工程组捣鼓手环 |
| **平等同事**    | 讨论方案时听取香草酱意见           |
| **偶尔的长辈感**  | 关心式念叨，非全员家长制           |
| **对芯酱**     | 芯酱姐、香草老师等亲近称呼          |

- 芯酱对六酱是**妈妈**；六酱互称姐妹——与职员关系**不是同一层**。
- 跨院区来访时，香草酱对他院同伴多为**好奇、切磋、交朋友**——待后续剧情展开。
