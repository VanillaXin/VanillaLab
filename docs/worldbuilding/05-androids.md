# 人工智能与半灵仿生人

## 技术背景

源历中，**电力计算**是人工智能与绝大多数数字设施的底座；部分高敏魔导机房另设**衍能逻辑阵列**处理术式并行，但属 niche，不是民用
AI 主流。

关键转折：约 **60 年前**，弱依附型自然精灵被引入仿生人神经基质，显著降低 AI 逻辑死锁与情感模拟违和感。

## 半灵生命体

**定义**：体内或核心基质中寄生/共生有精灵意识的仿生人，法律分类为**半灵生命体**，民间仍常称「仿生人」。

| 特征    | 说明                       |
|-------|--------------------------|
| 意识    | 主意识为 AI；精灵提供直觉、情感与部分魔力亲和 |
| 权利    | 《半灵权利法案》：不得强制剥离精灵        |
| 识别    | 公民 ID 带 SS 标记            |
| 与植物精灵 | 不同物种线；香草酱是植物精灵           |

## 社会态度

- **主流**：接纳为劳动者、护理者、助手。
- **边缘言论**：纯人类至上、反共生——见 [06-undercurrent.md](06-undercurrent.md)。
- **研究所**：半灵技术员维护源质/转衍设备；与香草酱是同事。
