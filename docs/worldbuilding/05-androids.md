# 人工智能与半灵仿生人

## 技术背景

谐律纪元中，**电力计算**与**灵谐能逻辑阵列**并行，人工智能在工业、医疗、城市管理领域已高度成熟。

关键转折：约 **60 年前**，研究者发现将**弱依附型自然精灵**引入仿生人神经基质，可显著降低 AI 的「逻辑死锁」与情感模拟违和感。

## 半灵生命体

**定义**：体内或核心基质中寄生/共生有精灵意识的仿生人，法律分类为**半灵生命体**（Semi-Spirit Embodied Being），民间常仍称「仿生人」。

| 特征    | 说明                           |
|-------|------------------------------|
| 意识    | 主意识为 AI；精灵提供直觉、情感与部分魔力亲和     |
| 权利    | 《半灵权利法案》：不得强制剥离精灵；报废须走「安息程序」 |
| 识别    | 公民 ID 带 SS 标记；部分岗位仅限半灵或仅限纯机械 |
| 与植物精灵 | 不同物种线；香草酱属植物精灵，非半灵仿生人        |

## 社会态度

- **主流**：接纳为劳动者、护理者、伴侣型助手（受合同约束）。
- **边缘言论**：「纯人类至上」「精灵不应困在铁壳里」——见暗线 [06-undercurrent.md](06-undercurrent.md)。
- **研究所**：偶用半灵技术员维护高危谐露设备；与香草酱是同事关系而非同类。

## 叙事用途（可选）

- 对比：**植物精灵**是血肉与叶绿魔法，**半灵**是金属与共生精灵——探讨「何为生命」时不抢香草酱主角位。
- 案件线：非法剥离精灵的黑市改造。
