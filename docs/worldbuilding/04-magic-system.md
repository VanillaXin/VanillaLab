# 魔法体系

## 学科地位

魔法是谐律纪元下的**正规学科**，纳入国民教育与职业资格体系。核心原则：

> **可学、可考、可监管**——与「古代遗物无咏唱触发」的不可控性形成对立。

## 魔法使用资格考试

| 等级 | 俗称 | 权限概要              |
|----|----|-------------------|
| M0 | 见习 | 生活辅助类、无战斗/大型术式    |
| M1 | 初阶 | 登记职业魔法、小型防护       |
| M2 | 中阶 | 行业核心术式、部分储存装置授权   |
| M3 | 高阶 | 大型阵式、研究级实验（需单位挂靠） |
| M4 | 大师 | 极少数；多挂名研究所或议会专家组  |

- 考试由各国**魔导考试院**统一大纲，每年多轮。
- **无证施法**在公共场合属违法；紧急情况有「临时救助免责条款」但事后须报备。

## 施法方式

### 自行咏唱

- 依赖精神集中与咒文结构记忆。
- 植物精灵天生魔力亲和高，但仍建议考证以使用受监管的工业/医疗术式。

### 机械咏辅助

- 头戴式、手持式或房间级「咏唱增幅器」。
- 适合工程师、医师；香草酱中绿茶系倾向擅长与此类设备协同。

### 预储存装置

- 术式固化于晶片/卷轴匣，灵谐能预充。
- 需备案术式 ID；警用与军用版本不流入民间。

## 与古代遗物的区别

|    | 正规魔法     | 古代遗物     |
|----|----------|----------|
| 触发 | 咏唱/装置/资格 | 触碰或条件即发动 |
| 学习 | 可教学      | 不可复制原理   |
| 管理 | 考试与备案    | 官方回收、封存  |
| 风险 | 可控       | 极高、常不可逆  |

## 香草酱的魔法画像（概要）

| 角色 | 倾向（待细化）        |
|----|----------------|
| 初芯 | 培育系、结界、团队增幅    |
| 黑茶 | 防护、稳固、持久战式     |
| 红茶 | 温暖类治疗、士气类辅助    |
| 青茶 | 幻惑、气息隐匿、平衡型    |
| 黄茶 | 时间缓释类、精细控制（稀有） |
| 白茶 | 净化、愈伤          |
| 绿茶 | 机械咏协同、信息类小魔法   |

具体术式名可在角色篇逐条补充。
