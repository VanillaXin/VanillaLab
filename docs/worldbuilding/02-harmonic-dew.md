# 谐露

## 定名

**谐露**（Harmonic Dew）：半透明、略黏稠的液态物质，自然存在于部分灵脉节点与特定植物组织内。

> 原稿「???」液态基础物质已定为此名。口语可说「露」「谐露剂」。

## 物理与魔法性质

| 性质 | 说明                              |
|----|---------------------------------|
| 吸魔 | 可缓慢吸收环境中的游离魔力，吸收率与纯度、温度相关       |
| 稳放 | 储存后按可控速率释放，波动远小于裸咏唱             |
| 外观 | 淡金至淡绿荧光，静置时有细微「呼吸」式明暗           |
| 危害 | **泄露**时魔力失控，可引发设备熔毁、生物畸变或小型「露灾」 |

## 用途

1. **工业**：谐露精炼后输入转换炉，产出灵谐能。
2. **医疗**：稀释谐露用于魔法创伤的稳定治疗（需持证医师操作）。
3. **农业**：精灵培育基质、大型温室的魔力缓冲液。
4. **研究所**：香草芯项目的精灵之种孵化液主要成分。

## 储存与运输

- **标准容器**：「谐封罐」——多层符文玻璃 + 灵谐能负压锁；罐体有颜色编码（民用黄、科研蓝、军用红）。
- **法规**：跨境运输需谐律议会配额；私人持有超过阈值需登记。
- **黑市**：稀释谐露、来路不明的「野露」是地下经济重要商品（暗线要素）。

## 与植物精灵的关系

植物精灵体内有天然谐露循环；她们既是谐露生态的**受益者**，也是研究所眼中的**活体传感器**——可预警局部露浓度异常。这也是香草酱被委以野外监测任务的原因之一。
