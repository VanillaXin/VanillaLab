# 世界观总览

## 一句话

**源历**里，市民**插电过日子**，源质与衍能在设施深处默默转成电；魔法传了数百年却仍未吃透精髓。表面和平的日常之下，黑市源质、古代遗物与共生秩序的裂缝在暗处蔓延。

## 世界基调

| 维度   | 描述                                     |
|------|----------------------------------------|
| 表象   | 电车、终端、魔法考试、仿生人新闻——很像「真的很和平」的现代社会       |
| 能源   | **电**是日常语言；**衍能**是工厂里的中间环节；**源质**是战略资源 |
| 魔法   | 古老、可考、能用，**原理未完全掌握**；转电链路长期低效          |
| 里层   | 遗物黑市、野质交易、研究所政治                        |
| 与香草酱 | 植物精灵；设定不与现实品牌绑定                        |
