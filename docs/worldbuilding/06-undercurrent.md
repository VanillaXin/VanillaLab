# 暗流与悬念

> **写作约定**：本章只搭「暗流」框架，不定终局。标有 `【待展开】` 的条目供团队后续选用或废弃。

## 表象与里层的缝隙

公众相信：

- 源质事故已被技术驯服；
- 古代遗物正在减少；
- 人灵共生是进步共识。

而情报碎片显示：**平静是管理出来的结果**，不是世界本来如此。

## 势力与张力（框架）

### 1. 联议会 vs 地方利益

- 源质配额收紧时，部分城邦暗中放宽精炼产能。
- 香草芯**绿芯院区**依赖配额，亦可能被卷入外部游说；**与其他院区仍和睦**。【待展开】

### 2. 遗物管理局与黑市

- 官方机构 **「遗物统辖局」**（简称统辖局）回收、封存古代遗物。
- 黑市流通「触发型」遗物碎片，买家包括恐怖分子、收藏家、以及**不知名研究所**。【待展开】

### 3. 「纯相派」

- 思潮：魔法应回归「纯咏唱」，科技应回归「纯机械」，反对源质—衍能混合路线。
- 公开演讲合法；地下分支涉嫌破坏转换炉。【待展开】

### 4. 精灵之种来源疑问

- 初代用「精灵之种」培育六位后来者；档案中种子的**一次采集地点**被涂黑。
- 若种子来自禁区或遗物核心，则绿芯计划的伦理叙事将动摇——**贴近香草酱的核心悬念**。【待展开】

### 5. 院所协作与总署例行工作

- 各香草芯研究所**总体和睦**：共享数据、互派交流、联合课题常见；配额按规章分配，**不搞勾心斗角**，也无挖角、打压式竞争。【待展开】
- **总署**审计与考核以**合规、安全、标准统一**为目的，例行、透明；职员与院区关系偏「系统管家」，不是权斗上位。【待展开】
- 若出现张力，多来自**技术路线分歧**或**资源紧张时的优先级讨论**，当面说清、对事不对人。

### 6. 质灾遗迹

- 裂相纪末的质灾区仍有异常魔力；有势力想在那里开采「野质」。
- 可安排野外章：茶系香草酱监测任务遇险。【待展开】

## 不会立刻打破和平的节奏

- 日常篇：考试、所内实验、城市节庆、仿生人邻居。
- 暗线每 **3～5 个日常单元** 投一次「异样细节」（新闻一句、档案缺页、陌生人注视）。
- 避免开局即世界末日；**老套路的破局感**来自「信任被侵蚀」而非纯爆炸。
