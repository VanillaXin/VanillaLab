# 香草 · 初代（芯酱）

## 基础信息

| 项    | 内容                            |
|------|-------------------------------|
| 代号   | **香草**                        |
| 所内简称 | **芯酱**                        |
| 茶系   | 香草精灵（非单一茶种，象征香草本身）            |
| 候选姓名 | 温芯 / 梵芯（**正式姓名待定**）           |
| 年龄感  | 外表约 20～24，实际存在约 8 年（源历 72 年起） |
| 身份   | 香草芯研究所绿芯实验室负责人；六位香草酱的创造者      |

## 外貌与象征（建议）

- 发色：浅金或米白，末梢微绿；象征新芽与花蕊。
- 服饰：研究所白袍 + 心形蕊纹胸针；私下偏爱简洁连衣裙。
- 精灵特征：背后半透明叶翼（不常展开），情绪波动时散发淡淡香草气息。

## 性格

- **对外**：沉稳、有条理、善于演讲，是研究所的「门面」。
- **对内**：对六人有强烈责任感，偶尔把压力独自吞下。
- **缺陷**：不擅长请求帮助；对「种子来源」话题会下意识回避。

## 能力方向

| 类型 | 内容                     |
|----|------------------------|
| 核心 | 精灵之种分化、培育阵、源质活体稳态      |
| 战斗 | 非主职；必要时展开结界保护全员        |
| 特殊 | 与六位酱伴存在血脉共鸣，可远程感知其源质紊乱 |

## 背景故事（草案）

源历 72 年，绿芯计划在共鸣区实验室凝聚出第一枚稳定植物精灵意识。她自苏醒起便携带不完整记忆碎片——只有「要守护这片绿」的执念。

源历 75 年起，她以自身魔力为引，分六次培育出茶系香草酱。她相信她们是「家人」，而非档案上的代号条目。随着统辖局审计加紧，她开始梦见涂黑的档案与陌生的古代符文。

## 台词气质示例

> 「香草酱们，不是样品，是这家研究所的心跳。」  
> 「和平很好啊。所以更要守住那些不让和平碎掉的小事。」

## 与其他角色

- 六酱称呼她：**母亲**（建议统一）。
- 她称呼六人：**酱伴**，或所内简称（绿茶酱、红茶酱等）。
