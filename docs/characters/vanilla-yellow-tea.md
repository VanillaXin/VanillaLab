# V-004 · 香草黄茶（薄暮）

## 基础信息

| 项   | 内容              |
|-----|-----------------|
| 编号  | V-004           |
| 茶系  | 黄茶（轻发酵，工艺稀有）    |
| 代号  | **薄暮**          |
| 候选名 | **缃宁** / **秋禾** |
| 诞生  | 谐纪 77 年         |
| 关键词 | 稀有、细腻、记录、黄昏感    |

## 外貌与象征

- 发色淡金偏黄，齐肩；瞳色浅褐。
- 服饰偏复古文书风，多口袋，装笔记本与采样笔。
- 气息：稻香、微甜，像傍晚田间。

## 性格

- 六人中**最安静**的一位，擅长倾听与记录。
- 对数据与文献有病态整洁要求；所内档案多经她手。
- 缺陷：社交能量低，大型活动后会「关机」一整天。

## 能力方向

| 类型 | 内容                      |
|----|-------------------------|
| 魔法 | 时间缓释类（术式持续长、爆发弱）、精细谐露测量 |
| 职务 | 档案与实验日志；伦理委员会书记员助理      |
| 特长 | 能发现档案缺页与矛盾脚注（种子之谜关键人）   |

## 小传

黄茶全球产量稀少，她的诞生也经历三次失败才成功。她自知「珍贵」却讨厌被当成吉祥物。薄暮常在黄昏时分独自校对日志，曾抄下一份被涂黑的种子来源坐标，尚未告诉初代。

## 关系速写

| 绿茶 | 帮绿茶整理闯祸报告      |
