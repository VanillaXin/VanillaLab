## 关系网（概要）

| 关系      | 描述                                      |
|---------|-----------------------------------------|
| 初代 ↔ 六人 | 创造者与被创造者；日常称「姐姐」或「老师」（由初代偏好决定，可统一为「芯姐」） |
| 黑茶 ↔ 绿茶 | 稳重 vs 冲动，常见吐槽组合                         |
| 红茶 ↔ 白茶 | 外向照顾与内向被照顾                              |
| 青茶 ↔ 黄茶 | 行动派与观察派，适合搭档外勤                          |
| 六人彼此    | 同期诞生，称「酱伴」；对外统一香草酱                      |
