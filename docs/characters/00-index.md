# 香草酱 · 角色索引

## 称呼体系

| 层级        | 说明              | 示例         |
|-----------|-----------------|------------|
| **统称**    | 对外与媒体           | 香草酱、香草酱们   |
| **代号**    | 登记用茶系名；初代为香草    | 绿茶、红茶、香草   |
| **所内简称**  | 同事日常            | 绿茶酱、红茶酱、芯酱 |
| **面对芯酱**  | 六酱统一认**妈妈**     | 妈妈、母亲      |
| **芯酱对孩子** | 称**孩子 / 孩子们**   | 「绿茶，过来」    |
| **姐妹之间**  | 按**外表**长幼       | 大姐、二姐…幺妹   |
| **姓名**    | 公民正式名，**另起、待定** | —          |

## 年龄说明（源历 80 年）

- **植物精灵**寿元漫长，**外表年龄感**常年停在青年段。
- **实际年龄**自苏醒/孕育起算；仅**黑茶酱**为**不详**。
- 姐妹称呼看**外表**，不看实龄（例：红茶外表最熟，实龄 43；绿茶外表最小，实龄 25）。

| 黑茶酱  | 2/29  | 约 23～24 | **不详**   | 源历 35    |
