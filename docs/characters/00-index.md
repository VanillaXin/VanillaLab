# 香草酱 · 角色索引

## 称呼体系

| 层级       | 说明                       | 示例         |
|----------|--------------------------|------------|
| **统称**   | 植物精灵角色对外与彼此的总称           | 香草酱、香草酱们   |
| **代号**   | 研究所登记用茶系标识（六茶=茶类名；初代=香草） | 绿茶、红茶、香草   |
| **所内简称** | 同事日常称呼，带「酱」有温度           | 绿茶酱、红茶酱、芯酱 |
| **姓名**   | 公民正式名字，**另起、待定**         | （尚未定名）     |

> 不再使用 `V-00x`、`绿芯·壹` 等冰冷编号。文书上写**代号**即可；所里叫人用**X酱**。

## 关系网（概要）

| 关系        | 描述                  |
|-----------|---------------------|
| 芯酱 ↔ 六酱   | 创造者与家人；六人称初代 **母亲** |
| 黑茶酱 ↔ 绿茶酱 | 稳重 vs 冲动            |
| 红茶酱 ↔ 白茶酱 | 外向照顾与内向被照顾          |
| 青茶酱 ↔ 黄茶酱 | 外勤搭档                |
| 六酱彼此      | **酱伴**              |
