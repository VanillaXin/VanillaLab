# 香草酱 · 角色索引

## 称呼体系

| 层级        | 说明              | 示例               |
|-----------|-----------------|------------------|
| **统称**    | 对外与媒体           | 香草酱、香草酱们         |
| **代号**    | 登记用茶系名；初代为香草    | 绿茶、红茶、香草         |
| **所内简称**  | 同事日常            | 绿茶酱、红茶酱、芯酱       |
| **面对芯酱**  | 六酱统一认**妈妈**     | 妈妈、母亲            |
| **芯酱对孩子** | 称**孩子 / 孩子们**   | 「绿茶，过来」「孩子们」     |
| **姐妹之间**  | 按外表长幼           | 大姐、二姐…幺妹；或直接叫 X酱 |
| **姓名**    | 公民正式名，**另起、待定** | —                |

研究所人类职员与香草酱：**同事、朋友**为主，相处融洽；个别人会留零食、念叨安全，但**没有**家人式客套或隔阂。对外仍用专业称谓。

## 谱系

```
                    ┌─────────────────┐
                    │ 香草 · 芯酱      │
                    │ 妈妈             │
                    └────────┬────────┘
         ┌──────────┬───────┼───────┬──────────┬──────────┐
         ▼          ▼       ▼       ▼          ▼          ▼
       黑茶        红茶      青茶      黄茶       白茶       绿茶
      大姐        二姐      三姐      四姐       五妹       幺妹
```

> 姐妹序按**外表长幼**与所内习惯；黑茶酱年龄不详，仍被称「大姐」。

## 关系网（概要）

| 关系       | 描述                   |
|----------|----------------------|
| 芯酱 ↔ 六酱  | **母子**；六酱称妈妈         |
| 六酱彼此     | **姐妹**；酱伴仍可用于姐妹互称    |
| 职员 ↔ 香草酱 | 朋友、搭档、同事；融洽，非隔阂式「家人」 |
