# V-005 · 香草白茶（初雪）

## 基础信息

| 项   | 内容              |
|-----|-----------------|
| 编号  | V-005           |
| 茶系  | 白茶（微发酵，芽叶）      |
| 代号  | **初雪**          |
| 候选名 | **素霏** / **银芽** |
| 诞生  | 谐纪 77 年         |
| 关键词 | 纯净、净化、新生、内向     |

## 外貌与象征

- 发色银白或极浅金，短发；瞳色淡灰蓝。
- 服饰白系为主，边缘霜纹；动作轻、脚步轻。
- 气息：清凉、洁净，近身可感微量净化效应。

## 性格

- 天真但不幼稚，对恶意敏感，像雪感知体温。
- 在医疗与净化任务中极度专注，平时却怕大声说话。
- 缺陷：容易自责，把团队失误揽到自己身上。

## 能力方向

| 类型 | 内容             |
|----|----------------|
| 魔法 | 净化、愈伤、谐露污染小幅中和 |
| 职务 | 医疗站精灵专员；孵化基质维护 |
| 弱点 | 过度净化会自身魔力透支    |

## 小传

她是六人中**外表最年幼**的一位，公民登记年龄却与青茶同期。红茶总给她带甜食；她悄悄把第一份留给初代。某次野露泄漏事故中，她强行净化成功而昏迷，引发公众对「香草酱是否被过度使用」的讨论——可作为社会议题篇。
