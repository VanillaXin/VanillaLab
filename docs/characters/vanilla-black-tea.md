# V-001 · 香草黑茶（沉韵）

## 基础信息

| 项   | 内容              |
|-----|-----------------|
| 编号  | V-001           |
| 茶系  | 黑茶（全发酵）         |
| 代号  | **沉韵**          |
| 候选名 | **玄默** / **墨苓** |
| 诞生  | 谐纪 75 年，第一批种子分化 |
| 关键词 | 稳重、防护、守夜、责任感    |

## 外貌与象征

- 发色深褐近黑，发尾暗红；瞳色琥珀。
- 服饰偏深色长外套，便于外勤；常佩谐封罐空罐作挂饰（提醒安全）。
- 气息：熟谷、木质，令人安心。

## 性格

- 团队里的**定海神针**，话不多但句句实在。
- 对规则与安全极其认真，有时会显得唠叨。
- 隐藏温柔：会默默记住每位酱伴的饮食禁忌。

## 能力方向

| 类型 | 内容                       |
|----|--------------------------|
| 魔法 | M2 方向；防护结界、谐露稳定术、持久战式魔力锁 |
| 职务 | 外勤监测队副队长；夜班谐露节点巡检        |
| 弱点 | 不擅表达情感；过载时会「沉默暴走」式加班     |

## 小传

诞生时谐露浓度偏高，身体一度不稳定，由初代彻夜维持阵式。康复后她立下规矩：「不让任何人再经历那种夜。」她主动申请最危险的巡检时段，绿茶常偷偷跟班——她表面训斥，实则默许。
