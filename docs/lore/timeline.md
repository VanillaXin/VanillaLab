# 简明年表

|---------|----------------------------------|

## 裂相纪 — 梗概

| 时期       | 事件                       |
|----------|--------------------------|
| 约 200 年前 | 源质工业开采                   |
| 约 150 年前 | 多次质灾                     |
| 约 120 年前 | 源质→衍能→**电**链路并网，**源历元年** |

## 植物精灵与绿芯院（长寿叙事）

> **植物精灵**寿元可达数百年；外表年龄感几乎不随岁月变化。  
> **绿芯院区**运作至今约 **68 源历年**；人类研究员、职员已轮换 **三至四代**，香草酱仍是院里的「老人」。
