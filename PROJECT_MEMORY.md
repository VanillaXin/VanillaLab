## 魔法与能源（重要）

- 魔法在**古法时代**已存在，进入源历后纳入考试；**精髓未完全掌握**。
- 源质 → 衍能 → 电 全链路**效率低**（约 35%～45%）。
- 写日常：聊**电**；写所内/工厂：聊**质、衍、转电效率**。
