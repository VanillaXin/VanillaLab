# 香草芯 · Vanilla Xin

香草芯团队的原创角色与世界观设定仓库。角色统称 **香草酱**。

## 叙事舞台

- **主叙事院所**：香草芯研究所·**绿芯院区**（芯酱与六名茶系香草酱）
- **上级机构**：香草芯研究总署（简称**总署**）
- **他院区**：滨海、北境等——预留扩展，见 [08-vanilla-core-network.md](docs/worldbuilding/08-vanilla-core-network.md)
